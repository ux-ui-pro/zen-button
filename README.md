## Features

- No dependencies
- Lightweight, 7.5Kb (2.7Kb gzipped)
- Flexible and extensible
- Types of buttons: text, outlined, contained
- Grouping different types of buttons
- Progress animation
- Opaque disabled
- Ready to go with icons
- Ripple effect
